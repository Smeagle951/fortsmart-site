Política de Privacidade

Última atualização: Janeiro 2026

1. Informações Gerais

O FortSmart Agro respeita sua privacidade e está comprometido em proteger seus dados pessoais. Esta política explica como coletamos, usamos e protegemos suas informações.

2. Dados Coletados

2.1. Dados Fornecidos pelo Usuário

Coletamos os seguintes dados quando você usa o aplicativo:

  • Informações de conta: nome, email, senha (criptografada)
  • Dados de fazendas: nome, localização, informações de propriedades rurais
  • Dados de talhões: informações geográficas, área, cultura, safras
  • Dados técnicos: registros de plantio, monitoramento, aplicações, custos
  • Fotos: imagens capturadas para documentação agronômica

2.2. Dados Coletados Automaticamente

  • Localização GPS: para georreferenciamento de registros (apenas quando você autoriza)
  • Dados de uso: informações sobre como você usa o aplicativo
  • Dados do dispositivo: modelo, versão do sistema operacional

3. Como Usamos Seus Dados

Usamos seus dados para fornecer e melhorar nossos serviços, processar e armazenar suas informações agrícolas, gerar relatórios e análises, sincronizar dados entre dispositivos, enviar atualizações e notificações (se autorizado) e prestar suporte técnico.

4. Armazenamento de Dados

4.1. Armazenamento Local

Seus dados são armazenados localmente no seu dispositivo usando SQLite. Isso significa que seus dados permanecem no seu dispositivo, o aplicativo funciona 100% offline, e você tem controle total sobre seus dados.

4.2. Sincronização (Opcional)

Se você optar por sincronizar seus dados, eles serão armazenados de forma segura em servidores, com conexões criptografadas (HTTPS) e proteção com segurança de nível empresarial.

5. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros, exceto quando você autoriza explicitamente, para cumprir obrigações legais ou para proteger nossos direitos e segurança.

6. Segurança

Implementamos medidas de segurança para proteger seus dados, incluindo criptografia de senhas, conexões HTTPS para sincronização, armazenamento local seguro e acesso restrito aos dados.

7. Seus Direitos

Você tem o direito de acessar seus dados pessoais, corrigir dados incorretos, solicitar exclusão de dados, exportar seus dados e revogar consentimento.

8. Contato

Para questões sobre privacidade, entre em contato: fortsmart.01@gmail.com

9. Lei Aplicável

Esta política é regida pelas leis brasileiras, especialmente a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018).
